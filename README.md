# 几分饱

一个适合手机使用的饱腹感记录网页。选择 1–10 分饱，可选填食物名称与热量；记录会自动使用手机当前时间，并在统计页绘制当天 7:00–24:00 的饱腹感曲线。

## 主要功能

- 1–10 分饱的蓝色到红色渐变选择器
- 十条等级提示词可在网页内自行编辑
- 食物名称、热量选填，当日热量自动求和
- 当日记录列表，可删除误操作记录
- 按日期查看折线图；手机横屏时自动铺满屏幕且无需滚动
- 数据保存在当前浏览器的 `localStorage` 中
- 支持离线访问和“添加到主屏幕”

## 发布到 GitHub Pages

1. 在 GitHub 新建一个仓库。
2. 把本目录内的所有文件上传到仓库根目录。
3. 打开仓库的 **Settings → Pages**。
4. 在 **Build and deployment** 的 Source 中选择 **GitHub Actions**。
5. 推送后等待 `Deploy static site to Pages` 工作流完成，然后用手机打开 Pages 地址。

网页记录不会进入 GitHub 仓库。它们仅保存在记录所用的手机和浏览器中；清理浏览器数据或更换手机会丢失已有记录。
